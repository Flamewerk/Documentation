# üî• What is flamewerk ?

Flamewerk is a template to help developers and enthousiasts build a variaty of webapplication with only knowing a little bit of extended markdown.  
It uses the Nuxt3 ecosystem together with TailwindCSS and a (curated) host of modules to develiver profressional progressive webapps without much coding.  
The template does allow you to code and extend with or without out of the box components pretty easily with a vue3 component driven workflow .
